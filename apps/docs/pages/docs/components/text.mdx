Text component is the used to render text and paragraphs.

## Type

Change text color with property `type`.

<Playground code={`
<>
  <Text p={true} type="secondary">Do not go gentle into that good night.</Text>
  <Text p={true} type="tertiary">Do not go gentle into that good night.</Text>
  <Text p={true} type="quaternary">Do not go gentle into that good night.</Text>
</>
`} />

## Weights

Change font weith with property `weight`.

<Playground code={`
<Text
  p={true}
  weight="bold"
>
  Do not go gentle into that good night.
</Text>
`} />

## Sizes

Change font size with property `size`.

<Playground code={`
<Text
  p={true}
  size="2xl"
>
  Do not go gentle into that good night.
</Text>
`} />
